# Firebase CRUD App

A Flutter application built using Firebase that demonstrates complete CRUD (Create, Read, Update, Delete) operations with authentication and Firestore integration.

## Features

- Firebase Authentication
- Login & Signup
- Forgot Password
- Add Data
- Update Data
- Delete Data
- Fetch Data from Firestore
- Firestore Database Integration
- Clean UI
- Responsive Design

## Tech Stack

- Flutter
- Dart
- Firebase Authentication
- Cloud Firestore
